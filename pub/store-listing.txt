Full description:
This app adds Everett Transit buses information to MonTransit.

This app provides the buses schedule.

Everett Transit buses serve Everett, Washington, United States.

Once this application is installed, the MonTransit app will display buses information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by Sound Transit.
http://www.soundtransit.org/Developer-resources/Data-downloads

This application is free and open-source:
https://github.com/mtransitapps/us-everett-transit-bus-android

This app is not related with Everett Transit and Sound Transit.
